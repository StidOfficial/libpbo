# libpbo
Library for manage PBO file of Bohemia Interactive.

## Prerequisites
yum :
```
yum install gcc-c++ make cmake libstdc++ libssl-dev
```

apt-get :
```
apt-get install build-essential make cmake libstdc++6 libssl-dev
```

## Build
```
cmake CMakeLists.txt
make
```

## Install
```
make install
```
